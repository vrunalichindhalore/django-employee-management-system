# Django Employee Management System

## Overview

A Django-based Employee Management System that allows administrators to manage employee records efficiently. The project includes authentication and complete CRUD (Create, Read, Update, Delete) functionality.

## Features

* User Authentication (Login & Logout)
* Add New Employee
* View Employee Records
* Update Employee Details
* Delete Employee Records
* Employee Profile Image Support
* Responsive User Interface

## Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite
* Git & GitHub

## Project Structure

* Employees/ – Employee management application
* TEMPLATES/ – HTML templates
* media/images/ – Employee profile images
* myproject/ – Django project configuration
* manage.py – Django management script

## CRUD Operations

* Create Employee
* Read Employee Information
* Update Employee Details
* Delete Employee Records

## How to Run

1. Clone the repository

2. Create a virtual environment

```bash
python -m venv env
```

3. Activate the virtual environment

```bash
env\Scripts\activate
```

4. Install Django

```bash
pip install django
```

5. Run migrations

```bash
python manage.py migrate
```

6. Start the development server

```bash
python manage.py runserver
```

7. Open in browser

```text
http://127.0.0.1:8000/
```
